Modi 초음파 센서를 활용해 출입을 구분하고 이를 바탕으로 특정 공간의 혼잡도를 파악하는 방식으로 진행함.

1. 도서관 세미나룸에서 7가지 시나리오로 데이터 생성
입장만 반복, 퇴장만 반복, 입퇴장 번갈아 등 7가지 시나리오를 만들어 각각 데이터를 만듦

2. 측정한 데이터를 바탕으로 개발팀 3명이서 각각 다른 아이디어로 입퇴장을 구분해 혼잡도를 측정하는 알고리즘 만듦

3. 나는 딥러닝 기반 RNN으로 입퇴장을 구분하기로 함.

4. 데이터 특징
초음파 센서는 초음파에 맞고 돌아온 거리를 반환함. 즉, 사람이 지나가면 초음파 센서가 측정한 거리가 줄어들다가 저점을 찍고 다시 원래 거리로 돌아옴. 입장시 멀리서부터 탐지되므로 기울기가 천천히 감소, 퇴장시 가까이서부터 탐지되므로 기울기 빠르게 감소하는 특징을 가짐.

5. 데이터 전처리
센서 데이터를 입장과 퇴장을 자르는 알고리즘 개발. 만든 알고리즘으로 입장과 퇴장의 차이가 있는지 그래프를 그려보았고 차이가 존재함을 확인함.

6. 모델링
tensorflow를 활용해 simple RNN 모델을 구축함. 단순한 모델에서 복잡한 모델로 층을 쌓아가며 최종적으로 train accuracy 71%, test accuracy 83% 달성함.

7. 모델 적용
측정한 데이터를 20단위로 잘라 모델에 집어넣었음.

8. 크롤링
한경관 메뉴를 웹에서 크롤링하는 코드를 만들어 웹앱에서 구현될 수 있도록 진행


8. 배운점
solid 원칙, 클린 코딩, 클래스의 중요성
주피터 노트북에서 함수를 생성하지 않고 코딩을 진행해서 나중에 적용하기가 힘들었음. 개발 처음부터 적용될 여러가지 상황을 고려해 대응할 수 있는 코드를 만들기

현실적으로 완벽하게 구현되는 코드를 짜기 위해 많은 시간과 노력이 필요하다.
